# Huawei LiteOS 黑客马拉松·北京站

## Description
2017年1月华为黑客松物联网大赛，基于树莓派的，利用Google的Tensorflow实现物体识别，多角度拍摄物体并在上位机上利用Python进行三维建模，并且会说话的智能小车🚙
